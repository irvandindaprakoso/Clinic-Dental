# Clinic-Dental

### Website Clinic 
------------------------------------------------------------------------------
Web yang digunakan untuk mengelola data pasien dan mempermudah dalam melakukan transaksi 

**Feature**

* Tambah/ edit User
* Transaksi

* MySQL
